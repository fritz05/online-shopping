# online-shopping

### Description

- Sample Online Shopping Application

### Tech/Libraries Used

- React
- Typescript
- Vite
- Chakra
- ESlint and Prettier
- React Icon
- Sweet Alert 2
- Yarn

### How to run
Steps:
1. install dependencies using `yarn install`
2. after the installation run `yarn dev`
3. if the compilation finishes successfully, press 'o' key on the terminal
